# Blinkit-Dashboard
Interactive Power BI dashboard analyzing Blinkit sales performance, customer behavior, and outlet insights for data-driven decision-making.
📊 Blinkit Sales Analysis Dashboard – Power BI Project
📝 Project Overview
This project presents an interactive Sales Analysis Dashboard for Blinkit, developed using Power BI to analyze sales performance, customer purchasing behavior, and outlet-level insights.

The main objective of this project is to convert raw business data into meaningful insights through data visualization and support data-driven decision-making.

🎯 Business Objectives
Analyze overall sales performance

Identify top-performing product categories

Understand customer purchasing patterns

Evaluate outlet-wise and location-wise sales performance

Study the impact of outlet size and establishment year on revenue

Monitor customer ratings and average sales

📌 Key Performance Indicators (KPIs)
💰 Total Sales

📦 Number of Items Sold

⭐ Average Customer Rating

📊 Average Sales per Outlet

📈 Dashboard Insights
Sales by Outlet Location

Sales by Product Category

Sales based on Outlet Size

Outlet Establishment Year Analysis

Customer Rating Distribution

Item Type Contribution to Total Revenue
